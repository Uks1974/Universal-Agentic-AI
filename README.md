Universal Agentic AI – Decision Support System

Universal Agentic AI is a Python–Streamlit–based Decision Support System that enables users to upload data files (Excel, PDF, Word, PowerPoint, Images), automatically analyze them using AI, interact with the data through natural language, and generate executive-level summaries and reports.

The system is designed for business, government, consulting, and research use cases, where quick understanding, structured analysis, and decision-ready outputs are required.

Key capabilities include:

Multi-file ingestion and parsing

AI-driven data understanding

Interactive “Chat with Data”

Brief analysis with risks and recommended actions

One-page executive summary generation

Export to Word, Excel, PowerPoint, PDF, and Image

User-provided API key (BYOK) with validation

Language selection and dark mode
